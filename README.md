# Description

php로 구현한 반려동물 정보 제공 및 공유 커뮤니티 사이트입니다.

http://linecom1234.dothome.co.kr/project/pages/main.php


# 주요 기능

- 회원가입, 로그인, 마이페이지 구현
- 게시판 기능 구현
- 블로그 기능 구현
- 카카오 맵 API와 공공데이터 API를 사용한 주변 동물병원, 동물약국 정보 제공
- OAuth를 사용한 카카오, 네이버 로그인 기능 구현
- 쿠키를 사용한 아이디 저장 기능 구현
- 세션을 사용한 로그인 유지 기능
